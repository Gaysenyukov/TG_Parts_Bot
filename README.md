**Telegram Parts Bot**

Telegram-бот для поиска автотоваров через ABCP API  
Позволяет искать товары по артикулу и бренду с отображением цены и сроков поставки.

---

## Стек технологий

- Python 3.12
- python-telegram-bot 20+
- aiohttp
- ABCP API
- Git / GitHub

---

## Возможности

- Поиск товаров по артикулу
- Фиксированный бренд (Grass - временно для тестирования бизнес-логики)
- Расчёт цены с наценкой
- Отображение сроков поставки
- Очистка и нормализация артикулов
- FSM-логика взаимодействия с пользователем

---

## Архитектура

Проект построен по принципу разделения ответственности:
- bot/ — UI и Telegram-взаимодействие
- api/ — работа с внешним API
- services/ — бизнес-логика
- utils/ — чистые функции без побочных эффектов

Такой подход упрощает поддержку и масштабирование проекта.
